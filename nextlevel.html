Raspberry Pi

Der Raspberry Pi ist ein Mini-Computer, der ursprünglich für Schüler und Studenten gedacht war, er ermöglicht den Erstkontakt zu Linux, Shell Scripting, Programmieren, Physical Computing und kann auch im Produktiveinsatz verwendet werden.

Docker

Docker ist ein beliebtes Open-Source-Tool, das eine portable und konsistente Laufzeitumgebung für Softwareanwendungen bietet. Docker verwendet Container als isolierte Umgebungen.
